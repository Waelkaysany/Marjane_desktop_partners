# Product Page

This folder contains the separated components for the product page.

## File Structure

- `index.php` - Main HTML/PHP file for the product page
- `product.css` - CSS styles specific to the product page
- `product.js` - JavaScript functionality specific to the product page
- `README.md` - This documentation file

## Features

- **16 Products Display**: Shows all 16 products in a responsive grid layout
- **Cart Integration**: Full shopping cart functionality with add/remove items
- **User Authentication**: Session management and partner data display
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Navigation**: Links to home page, equipment page, and cart
- **User Profile**: Parameters menu with user information and logout

## Dependencies

- `../assets/css/main.css` - Main CSS styles
- `../style.css` - Global styles
- `../recommendation-product.js` - Product data and rendering functions
- `../cart.js` - Cart system functionality
- `../assets/js/profile.js` - Profile menu functionality

## Navigation

- **PROMOTIONS** → `../home.php`
- **PRODUCTS** → `index.php` (current page)
- **EQUIPMENTS** → `../equipment/equipment.php`
- **Cart** → `../cart.php`

## Usage

Access the product page by navigating to `/product/` or clicking the "PRODUCTS" link in the navigation menu.
